Trends in biblical quotations

The heart of this site is a set of interactive visualizations which let you see the trend in the rate of quotations for any given verse in Chronicling America and Nineteenth-century Newspapers, then see examples of those quotations on the newspaper page at Chronicling America.

If you have a verse you want to look up directly, you can search for it below. (Note that the lists of verses below are rather selective, and so return verses with higher quality results. The verse search feature below returns many more verses, and the quality of the results for some verses may therefore be lower.) You can search for either the reference of the verse (e.g., “Genesis 1:1”) or the text of the verse in the King James Version (e.g., “In the beginning, God created the heaven and the earth”).
